# نسخه 1.0.1: پولیش ظاهر افزونه

این فاز روی یکپارچه‌سازی و حرفه‌ای‌تر کردن ظاهر افزونه تمرکز دارد و منطق رزرو، پرداخت، کیف پول و API را تغییر نمی‌دهد.

## بخش‌های تغییرکرده

- رنگ‌بندی پایه فرانت‌اند متعادل‌تر شد تا کنار سبز پزشکی، آبی اطلاع‌رسانی و کهربایی وضعیت‌های حضوری هم حضور داشته باشند.
- هدر عمومی آرشیو پزشکان، شمارنده نتایج و فرم فیلتر ظاهر واضح‌تر و responsiveتری گرفتند.
- کارت پزشک با نوار ظریف وضعیت، تصویر تمیزتر، badgeهای رنگی‌تر، فضای داخلی بهتر و hover state حرفه‌ای‌تر بازطراحی شد.
- تقویم رزرو و اسلات‌ها از حالت پراکنده به چیدمان منظم‌تر با انتخاب، hover و وضعیت‌های خواناتر رسیدند.
- فرم رزرو داخل یک پنل سبک قرار گرفت تا از اسلات‌ها جدا و قابل اسکن باشد.
- کارت پروفایل پزشک، تصویر پزشک، آمار هزینه‌ها و گالری مطب از نظر فاصله، سایه و hierarchy بهتر شدند.
- پنل بیمار و داشبورد پزشک/منشی در sidebar، brand mark، topbar، کارت‌های آماری و جدول‌ها ظاهر یکپارچه‌تری گرفتند.
- رسید نوبت padding و کنتراست بهتری دارد.
- فایل `admin.css` برای کارت‌ها، جدول‌ها، فیلترها، فرم‌ها، تقویم شمسی و media fields پولیش شد.

## نکات نگهداری

- این تغییرها فقط CSS و مستندات را هدف گرفته‌اند، به جز bump نسخه.
- radius کنترل‌شده 8px حفظ شد تا UI فرم‌محور و SaaS بماند.
- داده‌های حساس زمانی همچنان از REST بارگذاری می‌شوند و در HTML کش‌شده اولیه قرار نمی‌گیرند.
